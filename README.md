# React + NextJS + Tailwind CSS + Typescript

## Deps

Node 16
React 18
Next 13
Tailwind CSS 3

## Commands to know about:

To install dependencies:
```bash
$ npm install
```

To run the app: 
```bash
$ npm run dev
```

To lint code: 
```bash
$ npm run lint
```

To create a new app build:
```bash
$ npm run build
```
